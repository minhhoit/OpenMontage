# Documentary Montage Scripts

Put reusable briefs, narration drafts, act outlines, and shot lists here.

Example:

```yaml
topic: "What a city does before sunrise"
template: youtube_shorts
tone: elegiac
duration_seconds: 60
narration: piper
music: local
sources_allowed:
  - archive_org
  - nasa
  - wikimedia
  - coverr
end_tag: "THE DAY BEGINS BEFORE US."
beats:
  - "Empty roads under sodium light"
  - "Workers preparing shops"
  - "First trains and buses"
  - "Windows turning on one by one"
```
